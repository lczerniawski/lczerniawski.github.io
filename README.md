# lczerniawski — Personal Website

Minimal personal site built with Astro. Focused on a simple blog and static
pages, with MD/MDX content collections and RSS/sitemap support.

## Tech

- Astro
- Deno

## Quick start

```bash
# install dependencies
deno ci

# start dev server
deno task dev

# build
deno task build

# preview the built site
deno task preview

# run Astro CLI via Deno
deno task astro -- --help
```
